# SecretMessageApp

A simple Android application built during an App Development workshop. The project was created as a beginner exercise to understand the basics of Android development using Kotlin.

## Overview

SecretMessageApp allows users to enter and display messages through a simple interface. The project was developed mainly for learning purposes and helped me become familiar with Android Studio, Kotlin, layouts, and user interactions.

## Features

- Simple and clean UI
- Message input functionality
- Basic Android activity structure
- Built using Kotlin

## Tech Stack

- Kotlin
- Android Studio
- XML Layouts

## Purpose

This project was created while attending an App Development workshop. At the time, I was exploring Android development and learning the fundamentals of building mobile applications.

> Note: This is a beginner-level learning project and is not intended for production use.

## Project Structure

```
SecretMessageApp
├── app
├── gradle
├── build.gradle
└── settings.gradle
```

## Getting Started

### Prerequisites

- Android Studio
- JDK 17 or above

### Installation

1. Clone the repository:

```bash
git clone https://github.com/GargCodesT18/SecretMessageApp.git
```

2. Open the project in Android Studio.

3. Sync Gradle and run the app on an emulator or physical device.
